# 엑셀 템플릿 콘텐츠 생성 규칙

적용 대상: 콘텐츠A팀 — 엑셀 템플릿 결과물 (발주관리, 재고관리, 업무관리 등)

---

## 1. 기본 원칙

1. 결과물은 **실무자가 바로 복사해서 쓸 수 있는 수준**으로 완성한다.
2. 서사형 설명이 아니라 **구조 설명형**으로 작성한다.
3. 모든 시트, 열, 수식, 드롭다운은 빠짐없이 명시한다.
4. 추상적 설명 금지 — 셀 위치, 수식 전체, 선택 값 목록까지 구체적으로 작성한다.
5. 한영 병렬 구조 적용하지 않는다. 한국어 단일 작성한다.

---

## 2. 결과물 구조 (고정)

모든 엑셀 템플릿 결과물은 아래 섹션을 순서대로 포함한다.

1. **템플릿 개요** — 이 템플릿이 해결하는 문제, 사용 대상, 핵심 키워드
2. **시트 구성** — 시트 목록, 각 시트의 역할
3. **열 구성 정의** — 시트별 열 이름, 데이터 타입, 설명
4. **주요 수식 목록** — 수식명, 위치(셀 주소), 기능 설명, 실제 수식 예시
5. **드롭다운 및 조건부 서식** — 드롭다운 선택값 목록, 조건부 서식 색상 기준
6. **사용 가이드** — 단계별 입력 순서, 검수 포인트
7. **샘플 데이터** — 최소 5행, 실무 상황 기반

---

## 3. 섹션별 작성 기준

### 3.1 템플릿 개요

- 해결하는 문제를 구체적인 실무 상황으로 1~2문장 설명한다.
- 사용 대상(직무, 경험 수준)을 명시한다.
- 핵심 키워드는 템플릿 주제를 기준으로 도출한다.

### 3.2 시트 구성

- 시트 번호, 시트명, 역할을 표로 작성한다.
- 시트가 3개 미만이면 구성 이유를 설명한다.
- 시트 간 데이터 흐름(참조 관계)을 명시한다.

### 3.3 열 구성 정의

- 모든 시트의 모든 열을 빠짐없이 작성한다.
- 데이터 타입: 텍스트 / 숫자 / 날짜 / 드롭다운 / 수식 중 하나를 명시한다.
- 설명은 "이 열에 무엇을 입력하는가"를 기준으로 작성한다.

### 3.4 주요 수식 목록

- 사용된 수식 전체를 나열한다.
- 각 수식마다 아래 4가지를 포함한다:
  1. 수식명 (예: `SUMIFS`)
  2. 위치 (예: `대시보드!B2`)
  3. 기능 설명 (1문장)
  4. 실제 수식 예시 (예: `=SUMIFS(입력!D:D, 입력!C:C, "완료")`)

### 3.5 드롭다운 및 조건부 서식

- 드롭다운 선택값은 전체 목록을 나열한다.
- 조건부 서식은 색상 기준과 적용 조건을 구체적으로 명시한다.
  - 예: "완료 → 초록(#C6EFCE), 보류 → 빨강(#FFC7CE), 진행중 → 파랑(#BDD7EE)"

### 3.6 사용 가이드

- 단계별로 끊어서 작성한다.
- 메뉴 위치, 클릭 순서, 열리는 창 이름까지 포함한다.
- 중간 단계를 생략하지 않는다.
- 검수 포인트: 샘플 데이터와 실제 데이터가 분리되어 있는지 확인하는 항목을 포함한다.

### 3.7 샘플 데이터

- 최소 5행 작성한다.
- 실무 상황 기반으로 작성한다 (가상 수치 사용 가능).
- 진행률, 상태, 담당자 등 핵심 열은 반드시 채운다.
- 메모 열에는 실제 실무 문제 상황을 짧게 기재한다.

---

## 4. 절대 금지 사항

- 섹션 생략 금지
- 수식 예시 없이 수식명만 나열 금지
- 드롭다운 선택값 일부 생략 금지
- 한영 병렬 구조 적용 금지
- 블로그 글 서론/결론 형식 적용 금지
- Q&A 섹션 삽입 금지 (블로그 전용)
- 추상적 설명("적절히 입력", "상황에 맞게 사용") 금지

---

## 5. 분량 기준

- 섹션별 완결성을 우선한다. 글자 수 기준은 적용하지 않는다.
- 열 구성 정의와 수식 목록은 실제 템플릿에 포함된 모든 항목을 빠짐없이 작성한다.
- 샘플 데이터는 최소 5행이며 생략 표시(`...`) 금지.

---

## 6. 파일명 및 메타데이터 규칙

결과물 파일명 형식:

```
YYYY-MM-DD_[주제]_v[버전]_final.md
```

메타데이터 헤더 (결과물 최상단에 포함):

```
버전: v[번호]
상태: 완성
상태 ID: PD-YYYY-[번호]
생성 기준: EXCEL_TEMPLATE_SPEC.md
```

---

## 7. 콘텐츠A팀 적용 주제 예시

이 규칙이 적용되는 템플릿 유형:

- 발주관리 엑셀시트
- 발주정리표
- 업무관리 엑셀템플릿
- 재고관리 엑셀템플릿
- 거래처관리표
- 프로젝트 일정관리표
- 비용정산 엑셀시트

블로그 주제와 혼용하지 않는다.
